# 🎯 Mock Interview Agent

An AI-powered Mock Interview Platform designed to help students and job seekers prepare for technical and HR interviews through AI-generated questions, resume analysis, performance evaluation, and personalized feedback.

---

## 📌 Overview

Mock Interview Agent is a full-stack web application that simulates real interview experiences using Artificial Intelligence. It enables users to practice interviews, analyze resumes, receive AI-based feedback, and improve their confidence before actual interviews.

---

## ✨ Features

- 🔐 User Authentication (Login & Signup)
- 📄 Resume Upload and AI Resume Analysis
- 🤖 AI-Generated Technical & HR Interview Questions
- 🎤 Voice-Based Interview Responses
- 📊 AI Performance Evaluation
- 📝 Personalized Feedback and Score
- 🏆 Leaderboard
- 👤 User Profile Management
- 📈 Interview Progress Tracking
- 🌐 Responsive User Interface

## 🚀 Installation

### 1. Clone Repository

```bash
git clone https://github.com/DHARANITHA2326/Mock-Interview-Agent.git
```

### 2. Navigate to Project

```bash
cd Mock-Interview-Agent
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Configure Environment Variables

Create a `.env` file in the project root.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
MONGODB_URI=YOUR_MONGODB_CONNECTION_STRING
APP_URL=http://localhost:3000
```

### 5. Start Development Server

```bash
npm run dev
```

### 6. Build Project

```bash
npm run build
```

### 7. Start Production Server

```bash
npm start
```
